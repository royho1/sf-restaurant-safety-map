# SF Restaurant Safety Map

Interactive web application visualizing 30,000+ restaurant inspection records from SF DataSF. Explore food safety scores by restaurant, neighborhood, or proximity to your location.

## Tech Stack

- **Backend:** Python, Flask, SQLite, Pandas
- **Frontend:** React, Mapbox GL JS
- **Data Source:** [SF DataSF - Restaurant Scores](https://data.sfgov.org/Health-and-Social-Services/Restaurant-Scores-LIVES-Standard/pyih-qa8i)

## Features

- Interactive map with color-coded health score pins
- Search and filter by name, neighborhood, score range, cuisine type
- Restaurant detail panels with violation history timelines
- Neighborhood aggregate statistics
- "Near me" geolocation-based filtering
- Citywide stats and trends
